- Google Cloud Vision API（ https://cloud.google.com/vision/?hl=ja )を用いてOCRを実行
- なお現在は直接APIにアクセスしているわけではなく、手動でAPIにアクセスしてJSONデータを持ってきているので、今後はこの手間をなくしていくつもり
